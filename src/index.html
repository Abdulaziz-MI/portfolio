Projects

Kalimah

HTML | CSS (Bootstrap) | JS | APIs

Kalimah is an educational app that allows users to read, listen, and ponder over Arabic verses from the Quran and scholarly explanations of those verses.

The user is presented with a random verse and is expected to fill in the missing word from the translation using a text input. A scholarly explanation of the verse is provided after the correct answer is submitted. The project aims to add new features like multiple choice mode, user login, daily challenges, and user preferences.

Gems for the Heart

HTML | CSS (Tachyons) | ReactJS | APIs

Gems for the Heart is an app that compiles heart-softening prophetic narrations collected from Imam Al-Bukhari's compilation of books, Sahih Al-Bukhari. The app is intended to help Muslims soften their hearts as a preparation for Ramadan. Ramadan is a special time for Muslims as it is the month of the Quran and worship.
